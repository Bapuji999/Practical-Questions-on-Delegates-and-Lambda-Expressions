Practical Questions on Delegates, Lambda Expressions, and More

  • 1. Define a structure Circle with properties Radius and a method to calculate the area. Use a List<Circle> to calculate the areas of all circles using LINQ’s Select.
  • 2. Create a multicast delegate Notifier that sends messages via email and SMS. Test it by invoking both methods through the delegate.
  • 3. Write a program using a lambda expression to calculate the square root of a number and display the result.
  • 4. Create a partial class Vehicle with properties and a method Start. Extend this class in another file to include a Stop method and use both methods in a program to control a vehicle.
  • 5. Use a statement lambda to generate the first N Fibonacci numbers and print them.
  • 6. Write an anonymous method that checks if a given string contains the word “Lambda”.
  • 7. Create a List<int> and use a delegate with the Sort method to sort it in descending order.
  • 8. Use a lambda expression with Where to filter out all even numbers in a List<int>.
  • 9. Define a delegate Operation to add, subtract, multiply, and divide two numbers. Test this delegate by invoking all operations on two numbers.
  • 10. Write a static anonymous function to calculate the power of a number and display the result.
  • 11. Create a structure Point with X and Y properties. Use a List<Point> to find all points lying within a radius of 10 units from the origin.
  • 12. Use a statement lambda with GroupBy to group a List<Employee> by their department.
  • 13. Write a program to use multicast delegates for logging different events (e.g., writing to the console, logging to a file).
  • 14. Define a delegate ProcessNumber to find the sum of all elements in a List<int> using the ForEach method.
  • 15. Write a program using a lambda expression with OrderByDescending to sort a List<int> in descending order.
  • 16. Use an anonymous method with Exists to check if a List<string> contains the word "Lambda".
  • 17. Define a structure Complex for complex numbers and write methods to add and subtract two complex numbers.
  • 18. Create a partial class Student with properties Name and Marks in one file. In another file, extend the class to include a method IsPassing and use LINQ to filter out passing students in a List<Student>.
  • 19. Write a program that uses a static anonymous function with Where to filter a List<string> for items longer than 5 characters.
  • 20. Use a delegate with the FindAll method to find all numbers greater than 10 in a List<int>.
  • 21. Write a lambda expression with RemoveAll to remove all numbers divisible by 5 from a List<int>.
  • 22. Create a multicast delegate ListAction that:
    • Prints all items in a List<int>.
    • Finds and prints the maximum value.
  • 23. Use a statement lambda with ForEach to print the sum of ASCII values of all characters in each string in a List<string>.
  • 24. Define a structure Rectangle with properties Length and Breadth and write a method to calculate the area. Use a List<Rectangle> to calculate the areas of all rectangles.
  • 25. Write an anonymous method to calculate the factorial of a number and print the result.
  • 26. Create a partial class Student with a method to calculate the grade based on marks. In another file, create a List<Student> and print the grade for each student.
  • 27. Use a lambda expression to check if a given number is a prime number.
  • 28. Write a program to demonstrate how exceptions in one method of a multicast delegate affect subsequent calls.
  • 29. Use a delegate to perform multiple actions on a list, such as printing all numbers and calculating the average, for a List<int>.
  • 30. Write a statement lambda that reverses a string and prints the reversed string.
  • 31. Create a structure Car with properties like Model and Year. Write a method to display car details and use it with a List<Car>.
  • 32. Combine delegates, anonymous methods, and lambda expressions to calculate the sum, product, and average of a given list of numbers.
  • 33. Write an anonymous method with the Find method to find the first number greater than 20 in a List<int>.
  • 34. Create a static lambda to format a date string into a specific format (e.g., "yyyy-MM-dd").
  • 35. Write a program that uses multicast delegates and lambda expressions to process a user-entered string through multiple transformations (e.g., reverse, uppercase, length calculation).
  • 36. Use a lambda expression with Select to extract and print the first letter of each word in a List<string>.
